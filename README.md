# Report of the fifth INTOXICOM Workshop

This workshop was held in Athens, GR, in September 2025.

## Previewing the PDF

### Option 1

1. go to https://preview.biohackrxiv.org/
2. copy/paste `https://github.com/TGX-UM/INTOXICOM_Workshop_2/` in the "Repository:" field
3. Click the "Generate PDF" button and wait a few seconds
5. (Optionally) Download the PDF

### Option 2

1. go to https://github.com/TGX-UM/INTOXICOM_Workshop_2/actions
2. click a recent PDF generation job
3. under Artifacts, click "PDF and info.txt" to download a ZIP file with these two files
4. extract the PDF from the ZIP file
